# Prática – Como usar

1. **Copie** trechos para `functions.php` do tema (ou tema filho), ou
2. **Plugin:** cole o conteúdo de `05-plugin-demo-standards.php` em `wp-content/plugins/estudos-wp-fundamentos/estudos-wp-fundamentos.php` e ative.

**Teoria rápida:** no topo de cada `.php` há um bloco **REFERÊNCIA RÁPIDA** com a sintaxe daquele tópico. Tudo em uma página: [../REFERENCIA-RAPIDA.md](../REFERENCIA-RAPIDA.md).

**Não** use funções WordPress em `wp-config.php`. Em desenvolvimento, ative `WP_DEBUG`.

**Arquivos 07–10:** transações $wpdb (07), filters em cascata (08), estrutura/constantes (09), tabela de hooks de bootstrap (10).
